## 🔧 Como executar o projeto

```bash
# Clone o repositório
git clone https://github.com/luanargolodev/codelandia.git

# Entre no diretório
cd codelandia

# Escolha o seu desafio
cd desafio-

```

Após entrar no desafio escolhe, rode o servidor com o Live Server.
